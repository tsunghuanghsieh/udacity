# Bookshelf

## Set up database
From `bookshelf` folder, run `./setup.sh` which will create `bookshelf` and `bookshelf_test` database, initial mock data in the databases.

## Tear down database
From `bookshelf` folder, run `./teardown.sh`.

## Start the flask app
From `bookshelf` folder, run `./runflask.sh`.
